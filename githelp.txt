1. First check git installes or not
cmd : git 

2. Clone repository to localhot
cmd : git clone <HTTPS URL>

3. Check changes to the files
cmd : git status

4. Check current branch
cmd : git branch

5. Stage all untracked or modified files
cmd : git add -A

6. Put commit message
cmd : git commit -m "----"

7. Push to git
cmd : git push origin <Branch Name>

8. Create a new branch
cmd : git checkout -b <Branch Name>

9. Swithch to another branch
cmd : git checkout <Branch Name>

10. Check commit history
cmd : git log --oneline

11. Merge with master
cmd : git merge origin master
